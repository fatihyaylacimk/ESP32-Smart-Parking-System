# ESP32 Smart Parking System

This project implements a smart parking management system using ESP32.

The system automatically detects vehicle occupancy using ultrasonic sensors, controls gate access through RFID authentication, displays parking availability on an LCD screen, and provides visual indicators with LEDs.

## Features

- RFID access control
- Automatic gate opening and closing
- Servo motor integration
- Parking slot occupancy detection
- Ultrasonic sensor monitoring
- LCD parking status display
- LED status indicators
- Buzzer notifications
- Real-time parking management

## Components

- ESP32 DevKit
- MFRC522 RFID Reader
- HC-SR04 Ultrasonic Sensors
- Servo Motor
- LCD I2C Display
- LEDs
- Buzzer
